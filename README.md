# Mascoadopta 
### Funcionalidad
Aplicación pensada para adoptar mascotas y verlas publicadas en la aplicación junto las mascotas del resto de usuarios.

## Firebase
#### FirebaseAuth
Tiene implementado la función de FirebaseAuth para la autenticación y registro en la app.

#### Firebase Storage
Con la función de firebase storage, guardamos las mascotas adoptadas junto su imagen, nombre y dueño de la mascota

## Api Fotos
Tiene la aplicación una api añadida que accede a internet para obtener imagenes de perros de 
manera aleatoria para luego poder guardar esa mascota en el Firebase Storage.
